# SpeedBoost - Typing Speed Test

A modern web application to test and improve your typing speed with real-time feedback and competitive leaderboards.

## Features

- ⚡ Real-time WPM & CPM calculation
- ⏱️ 60-second countdown timer
- 🎯 Accuracy tracking with visual feedback
- 📝 15 diverse 100-word texts
- 🏆 Top 5 leaderboard system
- 📱 Responsive design for all devices
- 🎨 Modern UI with glassmorphism effects

## Quick Start

1. Clone the repository
2. Install Flask: `pip install flask`
3. Run: `python app.py`
4. Open: http://localhost:5000

## Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Design**: Modern gradients, glassmorphism
- **Storage**: JSON for leaderboard data

## How to Use

1. Enter your name
2. Click "Start Test"
3. Type the displayed text for 60 seconds
4. View your score on the leaderboard
5. Try again to beat your record!
